# 2D平台跳跃游戏实现计划

## 游戏核心功能

1. **玩家角色**：简单的彩色方块，支持左右移动和跳跃
2. **物理系统**：重力、碰撞检测、跳跃机制
3. **平台系统**：地面和浮动平台
4. **游戏目标**：收集物品或到达终点
5. **AI编程接口**：提供简单的控制函数，让小孩可以编写AI逻辑

## 技术实现

* 使用纯HTML5 Canvas进行渲染

* 纯JavaScript实现游戏逻辑，无外部依赖

* 单文件结构，便于理解和修改

* 清晰的代码注释，解释关键概念

## 关键实现细节

1. **物理系统**：

   * 重力加速度：0.5px/帧²

   * 跳跃力度：-15px/帧

   * 左右移动速度：5px/帧

2. **碰撞检测**：

   * 轴对齐 bounding box (AABB) 碰撞检测

   * 处理玩家与平台的碰撞

3. **AI编程接口**：

   * 提供`aiControl(player, platforms, coins)`函数

   * 玩家可以通过修改此函数来控制角色

   * 内置简单的示例AI

4. **游戏元素**：

   * 玩家：蓝色方块

   * 平台：绿色矩形

   * 金币：黄色圆形

   * 终点：红色旗帜

## 游戏流程

1. 玩家出生在起点
2. 收集金币获得分数
3. 到达终点进入下一关
4. AI通过分析游戏状态（玩家位置、平台位置、金币位置）来控制角色移动和跳跃

## 学习目标

* 学习碰撞检测原理

* 掌握简单的AI决策逻辑

* 培养问题解决能力
