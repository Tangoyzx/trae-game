# 游戏类文档

## 1. 概述

`classes/`目录包含游戏中所有可交互对象的类定义，包括玩家、平台、金币、终点和敌人。每个类负责管理自己的状态、行为和渲染。

## 2. 玩家类 (Player)

### 2.1 描述

`Player`类表示游戏中的玩家角色，负责处理玩家的移动、跳跃、碰撞检测和状态更新。

### 2.2 构造函数

```javascript
new Player(x, y)
```

| 参数 | 类型 | 描述 |
|------|------|------|
| `x` | 数值 | 玩家初始X坐标 |
| `y` | 数值 | 玩家初始Y坐标 |

### 2.3 属性

| 属性名 | 类型 | 描述 |
|--------|------|------|
| `x` | 数值 | 玩家当前X坐标 |
| `y` | 数值 | 玩家当前Y坐标 |
| `width` | 数值 | 玩家宽度，默认32像素 |
| `height` | 数值 | 玩家高度，默认32像素 |
| `velocityX` | 数值 | 玩家水平速度 |
| `velocityY` | 数值 | 玩家垂直速度 |
| `onGround` | 布尔值 | 玩家是否在地面上 |
| `score` | 数值 | 玩家当前分数 |
| `currentAnimation` | 字符串 | 当前动画状态，如'idle'、'walk'、'jump' |

### 2.4 方法

#### `draw(ctx)`
绘制玩家到画布上，包括动画效果。
- **参数**：`ctx` - Canvas 2D上下文
- **返回值**：无

#### `update(deltaTime)`
更新玩家状态，包括位置、碰撞检测和分数计算。
- **参数**：`deltaTime` - 时间差（毫秒）
- **返回值**：无

#### `updateAnimation()`
根据玩家的物理状态自动更新动画。
- **参数**：无
- **返回值**：无

#### `moveLeft()`
使玩家向左移动。
- **参数**：无
- **返回值**：无

#### `moveRight()`
使玩家向右移动。
- **参数**：无
- **返回值**：无

#### `jump()`
使玩家跳跃，只有在地面上时才有效。
- **参数**：无
- **返回值**：无

#### `stop()`
停止玩家的水平移动。
- **参数**：无
- **返回值**：无

#### `reset()`
重置玩家状态到初始值，包括位置、速度和动画。
- **参数**：无
- **返回值**：无

#### `checkCollision(object)`
检测玩家与其他对象的碰撞。
- **参数**：`object` - 要检测碰撞的对象
- **返回值**：布尔值，表示是否发生碰撞

#### `getScore()`
获取玩家当前分数。
- **参数**：无
- **返回值**：数值，表示玩家当前分数

## 6. 敌人类 (Enemy)

### 6.1 描述

`Enemy`类表示游戏中的敌人，玩家碰到敌人会死亡。

### 6.2 构造函数

```javascript
new Enemy(x, y)
```

| 参数 | 类型 | 描述 |
|------|------|------|
| `x` | 数值 | 敌人初始X坐标 |
| `y` | 数值 | 敌人初始Y坐标 |

### 6.3 属性

| 属性名 | 类型 | 描述 |
|--------|------|------|
| `x` | 数值 | 敌人当前X坐标 |
| `y` | 数值 | 敌人当前Y坐标 |
| `width` | 数值 | 敌人宽度，默认30像素 |
| `height` | 数值 | 敌人高度，默认30像素 |
| `velocityX` | 数值 | 敌人水平速度 |
| `velocityY` | 数值 | 敌人垂直速度 |
| `onGround` | 布尔值 | 敌人是否在地面上 |
| `patrolSpeed` | 数值 | 巡逻速度，默认1 |
| `chaseSpeed` | 数值 | 追击速度，默认3 |
| `alertRange` | 数值 | 警戒范围，默认150像素 |
| `direction` | 数值 | 巡逻方向，1向右，-1向左 |
| `isChasing` | 布尔值 | 是否在追击玩家 |

### 6.4 方法

#### `draw(ctx)`
绘制敌人到画布上。
- **参数**：`ctx` - Canvas 2D上下文
- **返回值**：无

#### `update(platforms, player)`
更新敌人状态，包括移动、巡逻和追击逻辑。
- **参数**：
  - `platforms` - 平台数组
  - `player` - 玩家对象
- **返回值**：无

#### `checkCollision(object)`
检测敌人与其他对象的碰撞。
- **参数**：`object` - 要检测碰撞的对象
- **返回值**：布尔值，表示是否发生碰撞
