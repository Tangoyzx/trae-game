# 配置模块文档

## 1. 概述

`config.js`模块包含游戏中使用的所有配置常量，便于集中管理和修改游戏参数。

## 2. 配置常量

| 常量名 | 类型 | 默认值 | 描述 |
|--------|------|--------|------|
| `CANVAS_WIDTH` | 数值 | 800 | 游戏画布宽度，单位：像素 |
| `CANVAS_HEIGHT` | 数值 | 600 | 游戏画布高度，单位：像素 |
| `GRAVITY` | 数值 | 0.5 | 重力加速度，影响角色下落速度 |
| `JUMP_FORCE` | 数值 | -15 | 跳跃力，负值表示向上跳跃 |
| `MOVE_SPEED` | 数值 | 5 | 角色移动速度，影响左右移动的速率 |

## 3. 使用示例

```javascript
import { GRAVITY, JUMP_FORCE } from './config.js';

// 应用重力
this.velocityY += GRAVITY;

// 执行跳跃
this.velocityY = JUMP_FORCE;
```

## 4. 扩展建议

- 添加新的配置常量时，遵循现有的命名规范
- 所有游戏中可调整的参数都应放在此文件中
- 考虑添加难度级别配置，允许动态调整游戏参数
