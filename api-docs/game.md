# 游戏主逻辑文档

## 1. 概述

`game.js`模块包含游戏的核心逻辑，包括游戏初始化、状态管理、主循环和对象更新。它是连接所有其他模块的中心。

## 2. 游戏类 (Game)

### 2.1 描述

`Game`类是游戏的核心控制器，负责初始化游戏对象、管理游戏状态和运行游戏主循环。

### 2.2 构造函数

```javascript
new Game()
```

- **参数**：无
- **返回值**：Game实例

### 2.3 属性

| 属性名 | 类型 | 描述 |
|--------|------|------|
| `canvas` | HTMLCanvasElement | 游戏画布元素 |
| `ctx` | CanvasRenderingContext2D | Canvas 2D上下文 |
| `scoreElement` | HTMLElement | 分数显示元素 |
| `gameState` | 字符串 | 当前游戏状态，可选值：'running', 'paused', 'dead' |
| `keyboardController` | KeyboardController | 键盘控制器实例 |
| `aiController` | AIController | AI控制器实例 |
| `player` | Player | 玩家对象 |
| `platforms` | 数组 | 平台对象数组 |
| `coins` | 数组 | 金币对象数组 |
| `goal` | Goal | 终点对象 |
| `enemies` | 数组 | 敌人对象数组 |

### 2.4 方法

#### `init()`
初始化游戏，设置Canvas尺寸，初始化游戏对象并启动主循环。
- **参数**：无
- **返回值**：无

#### `initGameObjects()`
初始化或重置游戏对象，包括玩家、平台、金币、终点和敌人。
- **参数**：无
- **返回值**：无

#### `setGameState(state)`
设置游戏状态。
- **参数**：`state` - 游戏状态字符串，可选值：'running', 'paused', 'dead'
- **返回值**：无

#### `restartGame()`
重启游戏，重置游戏状态和所有游戏对象。
- **参数**：无
- **返回值**：无

#### `updateScore()`
更新分数显示。
- **参数**：无
- **返回值**：无

#### `gameLoop()`
游戏主循环，负责更新游戏状态、绘制游戏对象和请求下一帧动画。
- **参数**：无
- **返回值**：无

## 3. 游戏状态

| 状态名 | 描述 |
|--------|------|
| `running` | 游戏正在运行，玩家可以控制角色 |
| `paused` | 游戏暂停，不更新游戏状态 |
| `dead` | 玩家死亡，显示死亡提示和重启按钮 |

## 4. 游戏流程

1. **初始化**：创建Game实例，加载Canvas，初始化游戏对象
2. **游戏主循环**：
   - 清空画布
   - 根据游戏状态执行不同逻辑
   - 更新玩家和敌人状态
   - 检测碰撞和游戏事件
   - 绘制游戏对象
   - 请求下一帧动画
3. **游戏结束条件**：
   - 玩家掉出屏幕底部
   - 玩家与敌人碰撞
   - 玩家到达终点
